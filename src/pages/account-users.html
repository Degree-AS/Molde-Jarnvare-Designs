Rollene betyr

Admin
Kan bestille, godkjenne, og administrere brukere/adresser/innstillinger.

Kan bestille og se firma-historikk. Kan ikke endre brukere eller fakturaadresse.

Kan se ordrer og fakturaer, men ikke bestille. Typisk regnskap eller controller.
